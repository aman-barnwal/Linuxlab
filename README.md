# Linux Shell Scripting Labs (Lab 03 to Lab 11)

This repository contains all Linux Shell Scripting experiments completed as part of the Linux & Shell Scripting course at UPES.  
Each lab folder includes:
- Shell programs (.sh)
- Output screenshots (.png)

---

## Lab Structure

| Lab No. | Topics Covered |
|--------|----------------|
| Lab 03 | Basic Shell Scripts |
| Lab 04 | Conditional Statements |
| Lab 05 | Loops and Arithmetic Operations |
| Lab 06 | String and Pattern Operations |
| Lab 07 | Process Management and Basic Utilities |
| Lab 08 | Commands, Permissions and System Information |
| Lab 09 | File Handling and System Monitoring |
| Lab 10 | Functions and Modular Programming |
| Lab 11 | Advanced Shell Scripting (Menu System, CSV Handling, Database Query etc.) |
